# **App Name**: MattressPro CRM

## Core Features:

- Selling Form Automation: Automate form number generation, date defaults, total calculations, payment status updates, stock reduction, and expense recording.
- Buying Form Excel Import: Enable importing product data from Excel, automate cost distribution, and stock updates upon saving.
- Expense Tracking & Smart Costs: Implement unified expense tracking with automated monthly recurring costs (salary, rent, electricity) and categorize expenses for filtering and reporting.
- Warehouse Stock Management: Automate stock levels based on buying, selling, and internal transfer forms. Implement low-stock alerts and fast search capabilities. Provide stock reports and Excel export/import functionality.
- Data Analysis Dashboard: Display key metrics (total sales, collected cash, outstanding debts, net profit, stock value) with quick indicators (unpaid sales, low-stock products, top debt customers). Include optional charts and detail mode with filtering and Excel export.
- Role-Based User Permissions: Implement role-based permissions for admin, data manager, and salesman roles. Include data protections and access restrictions to ensure data security and appropriate user actions.
- Smart Suggestion Tool: Provide smart suggestions for the salesman when adding products, quantities or customers, based on historic trends and data available about those parameters, or recent similar successful orders.

## Style Guidelines:

- Primary color: Calming blue (#64B5F6) for trust and reliability.
- Background color: Light gray (#F0F4F8) for a clean and modern look.
- Accent color: Soft green (#81C784) for positive feedback and completed tasks.
- Headline font: 'Poppins', a geometric sans-serif.
- Body font: 'PT Sans', a humanist sans-serif; used for body text and information within tables/forms.
- Use clean, minimalist icons to represent different categories and actions within the CRM.
- Design a mobile-friendly layout that minimizes clicks and maximizes speed, with intuitive navigation and clear form designs.
- Use subtle animations and transitions to provide feedback and improve the user experience.
